# dotnetGen_mysql
.NETCore + Mysql 生成器

开发初期，它可以选择数据库表创建项目；

开发阶段，它可以即时更新项目中数据层 SDK，提供业务开发基础；

优势：
 * 1、根据主键、唯一键、外键（1对1，1对多，多对多）生成功能丰富的数据库 SDK；
 * 2、避免随意创建表，严格把控数据库，有标准的ER图；
 * 3、统一规范数据库操作类与方法，一条心堆业务；

部署方法：
 * 运行环境：.net 2.0+
 * 安装 ServerWinService 服务
 * 向开发者发送 MakeCode 生成器
